# Promethee Landing

Coming-soon landing page for Promethee.

## Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion (parallax + scroll animations)
- Lucide icons

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. Vercel deploys this automatically.
